HORIZON GLIDE is an infinite isometric arcade shooter for PICO-8. Pilot a hovering ship across procedurally generated terrain while fighting enemies, dodging bombs, and racing through checkpoints.

Controls

  • Arrow Keys - Move
  • X - Shoot
  • O - Deploy mines

What's New

v0.20

  • 60fps gameplay (up from 30fps)
  • Soft hover suspension and camera smoothing for a much better flight feel
  • Screen-space collisions - hits now register where the ship visually appears
  • Longer, more graceful ramp launches
  • Split-color text effect across all UI
  • Robot flinch and HUD shake on damage
  • Snappier floating text (Mario-style pop)
  • Bomb event rebalance (shorter, less punishing)

v0.19

  • 60fps rendering engine rewrite with optimized tile caching
  • Music channels protected from SFX interruption
  • Dynamic cache margin system (fixes terrain pop-in on settings changes)

v0.18

  • Defensive mines system
  • Bomb evasion event type
  • Sprite-based HUD with animated talking character
  • Full audio pass (menus, weapons, pickups, impacts)
  • Performance optimization for stable frame rate
  • Bug fixes for death detection and game restart

Features

  • Procedurally generated infinite worlds
  • Three event types: combat, circle race, bomb evasion
  • Isometric 3D perspective built in PICO-8
  • Multiple biomes with varying terrain elevation
  • Difficulty scaling and score tracking
